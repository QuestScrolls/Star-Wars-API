# SWAPI People Fetcher

A simple Python script to download a specified number of Star Wars characters from the [Star Wars API (SWAPI)](https://swapi.dev/).

## Features
- Fetches data from the `/people/` endpoint.
- Handles pagination automatically.
- Graceful error handling for network issues and invalid input.

## Requirements
- Python 3.6+
- `requests` library

Install dependencies:
```bash
pip install requests
